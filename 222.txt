Acerca de las cadenas f-strings

Las variables se incluyen entre llaves y la cadena debe usar el prefijo f.

Adem√°s de que las f-strings son menos detalladas que cualquier otra opci√≥n de formato, es posible usar expresiones entre llaves. Estas expresiones pueden ser funciones u operaciones directas. Por ejemplo, si quiere representar el valor 1/6 como un porcentaje con una posici√≥n decimal, puede usar directamente la funci√≥n round():

Para usar una expresi√≥n no es necesaria una llamada de funci√≥n. Cualquiera de los m√©todos de cadena tambi√©n son v√°lidos. Por ejemplo, la cadena podr√≠a aplicar un uso espec√≠fico de may√∫sculas y min√∫sculas para crear un t√≠tulo:

Un bucle while realiza una operaci√≥n mientras (while, en ingl√©s) una determinada condici√≥n es True. Puede usar un bucle while para lo siguiente:

Buscar otra l√≠nea en un archivo.
Comprobar si se ha establecido alguna marca.
Comprobar si un usuario ha terminado de introducir valores.
Comprobar si algo m√°s ha cambiado para indicar que el c√≥digo puede dejar de realizar la operaci√≥n.


Lo m√°s importante que se debe recordar al crear bucles while es asegurarse de que cambia la condici√≥n. Si la condici√≥n siempre es True, Python seguir√° ejecutando el c√≥digo hasta que el programa se bloquee.

La sintaxis de un bucle while es similar a la de una instrucci√≥n if. Proporcione una condici√≥n y el c√≥digo que quiere ejecutar mientras la condici√≥n sea True.

Un bucle while tiene tres partes importantes:

La palabra clave while, seguida de un espacio.

La condici√≥n que se va a probar. Si la condici√≥n es True, se ejecutar√° el c√≥digo dentro del bucle while.

El c√≥digo que se quiere ejecutar en cada iteraci√≥n, con sangr√≠a aplicada mediante espacios en blanco. Por ejemplo:

while condition:
    # code here

Veamos c√≥mo se puede crear c√≥digo para pedir a los usuarios que introduzcan valores y, despu√©s, permitirles usar done cuando terminen de introducirlos. En nuestro ejemplo, la entrada de usuario es la condici√≥n que se comprueba al principio del bucle while.

while user_input.lower() != 'done':
    user_input = input('Enter a new value, or done when done')

Tenga en cuenta que usa input para preguntar a los usuarios. Cada vez que los usuarios escriben un nuevo valor, cambian la condici√≥n, lo que significa que el bucle while se cerrar√° una vez que se haya escrito done.
En nuestro ejemplo, hemos usado lower para convertir la entrada a min√∫sculas, lo que permite una comparaci√≥n que no distingue may√∫sculas de min√∫sculas.
Puede usar la cadena reci√©n escrita como lo har√≠a con cualquier otra cadena capturada con input. Si quiere agregarla a una lista, puede usar c√≥digo similar al ejemplo siguiente:

# Create the variable for user input

# Create the list to store the values
inputs = []

# The while loop
while user_input.lower() != 'done':
    # Check if there's a value in user_input
    if user_input:
        # Store the value in the list
        inputs.append(user_input)
    # Prompt for a new value
    user_input = input('Enter a new value, or done when done')

Observe la instrucci√≥n if dentro del bucle while. Esta instrucci√≥n prueba si hay un valor de cadena dentro de user_input. Si el bucle while se ejecuta por primera vez, no hay ning√∫n valor, por lo que no hay nada que almacenar en inputs. Despu√©s de que se ejecute por primera vez, user_input siempre conserva el valor que el usuario ha escrito. Dado que while est√° probando para asegurarse de que el valor no es igual a done (la palabra que el usuario escribir√° para salir de la aplicaci√≥n), sabe que el valor actual es uno que puede agregar a la lista.
Es posible que conozca bien otros lenguajes de programaci√≥n que admiten un bucle do, lo que le permite realizar una prueba en la parte inferior del bucle. Python no proporciona un bucle do.

En Python, las listas pueden almacenar cualquier tipo de valor, como cadenas o n√∫meros:

Python tiene muchos tipos que se pueden recorrer en bucle. Estos tipos se conocen como iterables.

Las listas de Python son iterables y se pueden usar con un bucle for. Se usa un bucle for con objetos iterables que va a recorrer en bucle un n√∫mero conocido de veces, una vez para cada elemento del objeto iterable.

El bucle for es una instrucci√≥n con cinco partes importantes:

La palabra for, seguida de un espacio.
El nombre de la variable que quiere crear para cada valor de la secuencia (number).
La palabra in, entre espacios.
El nombre de la lista (countdown, en el ejemplo anterior) u objeto iterable que quiere recorrer en bucle, seguido de dos puntos (:).
El c√≥digo que quiere ejecutar para cada elemento del objeto iterable, separado por espacios en blanco anidados.
